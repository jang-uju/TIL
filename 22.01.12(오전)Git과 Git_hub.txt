Git 개념잡기
Git이란 소스코드관리 프로그램 -> 소스코드를 저장한다(소스코드 날리는 경우 방지)

git내 소스코드 '내 컴퓨터'에 저장 하는 것 - commit
내 소스코드를 '외부'에 있는 안전한 금고에 저장 - push

내 컴퓨터 저장소 : Local Repository
원격지 저장소 : Remote Repository
